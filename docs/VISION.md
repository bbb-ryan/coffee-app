# Product Vision

## The Problem

Coffee is overwhelming. There are hundreds of origins, roast levels, processing methods, and brewing techniques. Most people stick with what they know because exploring feels intimidating.

## The Solution

A simple, friendly app that meets people where they are — whether they're grabbing their first bag of whole beans or dialing in a pour-over. The app should feel like a knowledgeable friend, not a snobby barista.

## Core Principles

1. **Approachable over exhaustive** — Guide people, don't lecture them
2. **Personal over generic** — Learn what someone likes, then recommend accordingly
3. **Community-driven** — Real reviews and recommendations from real people
4. **Beautiful and tactile** — Coffee is sensory; the app should feel that way too

## Key Features (Planned)

### Phase 1 — Foundation
- [ ] Bean catalog with origin, roast, and flavor profiles
- [ ] Basic search and filtering
- [ ] Bean detail pages

### Phase 2 — Personalization
- [ ] Taste profile quiz
- [ ] Personal bean journal (tried / loved / want to try)
- [ ] Recommendations based on preferences

### Phase 3 — Community
- [ ] User reviews and ratings
- [ ] Shared lists and recommendations
- [ ] Brewing guides paired with beans

## Open Questions

- What tech stack? (leaning toward a web app to start, mobile later)
- Should we include a bean sourcing / purchasing component?
- How deep do we go on brewing guides?
