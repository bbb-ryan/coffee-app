# Prototypes

Design and concept exploration that hasn't earned a place in `src/` yet.

This folder replaces ad-hoc exploration directories like `hassan-design/`. Per `docs/WORKFLOW.md`, **a prototype is either absorbed into `src/` within a sprint, or moved to `archive/`**. It does not live here indefinitely.

## Convention

Each prototype is a folder:

```
prototypes/<name>/
├── README.md       # what it is, who made it, what it's testing
└── <files>         # HTML, reference .tsx, screenshots, etc.
```
